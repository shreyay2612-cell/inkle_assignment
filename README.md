Inkle Assignment: Frontend Intern

Overview:
This project is a frontend assignment for the Inkle internship position.  
It implements a pixel-perfect UI built with React and [@tanstack/react-table](https://tanstack.com/table/latest), matching the provided Figma design.  
The app displays data from public APIs in a table with edit and filter features, including a responsive modal for editing user details.

Features:
- Table UI: Built using [@tanstack/react-table], styled exactly to the Figma specification.
- Edit Modal: Update name and country with a modern dropdown; modal follows exact design specs.
- Country Filter: Multi-select dropdown lets you filter rows by country.
- Pixel-perfect: All fonts, colors, icons, spacing, corners, badges, and modal layouts match the provided design.
- Live API Integration: Data fetched from real APIs for demonstration.
- Responsive Design: Works smoothly on desktop.

Tech Stack
- React (CRA, JavaScript)
- @tanstack/react-table (table rendering)
- Material-UI (MUI) for modals, inputs, icons
- Axios for API calls

APIs Used
- [Taxes API](https://685013d7e7c42cfd17974a33.mockapi.io/taxes)
- [Countries API](https://685013d7e7c42cfd17974a33.mockapi.io/countries)
